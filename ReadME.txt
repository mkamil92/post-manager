=====================================Post Manager=======================================
Please follow below steps in order to exectute application successfully:
-> Unzip folder and cd post-manager
-> run following commands:
	- npm install
	- npm install bootstrap
	- npm install axios
	- npm install react-router-dom@5
-> once done all commands now it's time to execute applications sp run command:
	- npm start
